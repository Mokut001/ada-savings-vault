# ADA Savings Goal DApp (Timed Edition)

## Features
- **Smart Logic**: Automatically calculates monthly savings based on Start and Target dates.
- **Progress Notifier**: Visual bar updates and tells you days remaining.
- **Smart Contract**: Secures ADA until target amount is reached.

## Setup
1. Run `npm install`.
2. Add your Blockfrost Key to `.env.local`.
3. Run `npm run dev`.
